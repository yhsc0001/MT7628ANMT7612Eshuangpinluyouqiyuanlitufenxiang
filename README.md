## 方案简介

联发科的MT7628AN是一款高度集成的Wi-Fi SoC，适用于家用路由器和接入点，集成了580MHz MIPS 24KEc核心、以太网MAC/PHY以及Wi-Fi功能，支持802.11n标准。而MT7612E则是一个高性能的2x2 MIMO 802.11ac Wi-Fi芯片，专为提升无线网络速度和稳定性设计，支持高达867Mbps的数据传输速率。
